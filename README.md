## 📖 Project Description
This project implements important statistical methods used in data analysis:
- **Spearman Rank Correlation** → Measures relationship between two variables
- **Parabolic Curve Fitting** → Fits a quadratic curve using Least Squares Method

The application is built using **Python + Streamlit**, providing an interactive UI for performing calculations and visualizing results.

---

## 🚀 Features
- 📂 Upload CSV file or manually enter data
- 📊 Automatic rank calculation with tie handling
- 📈 Interactive graphical visualization (6 plots)
- 🧮 Step-by-step mathematical calculations
- 🎯 Clean and user-friendly interface
- ⚡ Real-time computation using Streamlit
